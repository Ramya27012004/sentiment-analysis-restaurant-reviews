#  Sentiment Analysis of Restaurant Reviews

##  Project Overview
This project focuses on analyzing customer reviews of restaurants and classifying them as **positive** or **negative** using Machine Learning techniques. It helps businesses understand customer feedback and improve their services.

---

## Objective
- To build a sentiment classification model using text data  
- To preprocess and clean real-world customer reviews  
- To apply Natural Language Processing (NLP) techniques  
- To evaluate model performance  

---

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  
    

---

## Features
- Data cleaning and preprocessing  
- Text tokenization and stopword removal  
- Feature extraction (Bag of Words / TF-IDF)  
- Model building using Naïve Bayes algorithm  
- Model evaluation (accuracy, predictions)
 ---
##  Results
- Successfully classified restaurant reviews into positive and negative sentiments  
- Achieved good accuracy using Naïve Bayes classifier
